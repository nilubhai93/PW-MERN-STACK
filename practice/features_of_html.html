Features of HTML5

1. Semantic Elements

HTML5 introduces several new semantic elements that provide better structure and meaning to web content. These elements help search engines and other user devices understand the content of web pages better.

  • <header>
  • <footer>
  • <section>
  • <article>
  • <nav>
  • <aside>

2. Multimedia Support

HTML5 provides native support for embedding audio and video content without the need for external plugins like Flash. This makes it easier to add multimedia content to web pages and enhances the user experience.

Example :

Video :


    <video controls >
        <source src="demo video.mp4" type="video/mp4">
    </video>

Audio:


    <audio controls>
        <source src="example.mp3" type="audio/mpeg">
    </audio>

3. Canvas API

The <canvas> element in HTML5 provides a powerful and flexible way to draw graphics using JavaScript. This element can be used for a wide range of purposes, including creating graphs, games, and interactive animations.

Example:


    <canvas id="myCanvas" width="200" height="200">
        Your browser does not support the canvas element.
    <canvas>
